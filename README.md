# markdown2unicode
Converter from markdown notation to unicode characters

## Formatting

| Markdown      | Unicode       |
|---------------|---------------|
| `**bold**`    | 𝗯𝗼𝗹𝗱          |
| `*italic*`    | 𝘪𝘵𝘢𝘭𝘪𝘤          |
| `***both***`  | 𝙗𝙤𝙩𝙝          |
| `` `mono` ``  | 𝚖𝚘𝚗𝚘          |
